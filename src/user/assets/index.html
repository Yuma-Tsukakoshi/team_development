ーAbout this websiteー

就活エージェントとは？：
就活のプロと対話しながら
それぞれの学生に合った企業を
探すサービスのこと

比較サイトとは？：
エージェントを絞り込み検索で
比較するサイト


このサイトでは、
数あるエージェントの中から
あなたに合ったエージェント選びを
サポートします。

使い方

ーHOW TOー

検索

自分の専攻・興味のあるエージェ
ントで絞り込み、検索してみる。

エージェントをクリック

気になったのを何個か選んで、カートに入れる。

(1度個人情報を登録すれば、エージェントの 一括申請が可能です)

応募

個人情報登録で応募完了。
後日、エージェントから連絡がきます。
