ーAbout this websiteー

就活エージェントとは？：
就活のプロと対話しながら
それぞれの学生に合った企業を
探すサービスのこと

比較サイトとは？：
エージェントを絞り込み検索で
比較するサイト


このサイトでは、
数あるエージェントの中から
あなたに合ったエージェント選びを
サポートします。

使い方

ーHOW TOー

探す

エージェントを絞り込み、
検索してみる。

選ぶ

気になったエージェントを何個か選んで、
カートに入れる。

(エージェントの 一括申請が可能です)

応募

個人情報登録で応募完了。
後日、エージェントから連絡がきます。
